# デプロイ前チェックリスト

## 必須項目

- [ ] GitHubリポジトリが作成されている
- [ ] コードがGitHubにプッシュされている
- [ ] Vercelアカウントが作成されている
- [ ] 本番環境用のSupabaseプロジェクトが作成されている（または開発環境を使用）
- [ ] 環境変数が準備されている

## 環境変数の確認

- [ ] `NEXT_PUBLIC_SUPABASE_URL`が設定されている
- [ ] `NEXT_PUBLIC_SUPABASE_ANON_KEY`が設定されている
- [ ] `OPENAI_API_KEY`が設定されている（使用する場合）

## Supabaseの設定

- [ ] データベーススキーマが適用されている
- [ ] ストレージバケットが作成されている
- [ ] RLSポリシーが正しく設定されている
- [ ] メール確認の設定を確認（本番環境では有効推奨）

## コードの確認

- [ ] `.env.local`が`.gitignore`に含まれている
- [ ] 機密情報がコードに含まれていない
- [ ] `npm run build`がローカルで成功する

## デプロイ後の確認

- [ ] アプリが正常に表示される
- [ ] ログイン・新規登録が動作する
- [ ] データの保存・読み込みが動作する
- [ ] エラーが発生しない
