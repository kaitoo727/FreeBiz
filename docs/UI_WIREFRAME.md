## 画面遷移図

```
ログイン
  ↓
オンボーディング
  ↓
ダッシュボード
  ├─→ 売上登録
  ├─→ 経費登録
  ├─→ 在庫管理（物販）
  ├─→ 案件管理（サービス）
  ├─→ 税金シミュレーター
  └─→ 確定申告書類
```

## レスポンシブデザイン

- モバイル: 1カラムレイアウト
- タブレット: 2カラムレイアウト
- デスクトップ: 3-4カラムレイアウト

## カラーパレット

- プライマリ: #0ea5e9 (Blue 500)
- 成功: #10b981 (Green 500)
- 警告: #f59e0b (Yellow 500)
- エラー: #ef4444 (Red 500)
- 背景: #f9fafb (Gray 50)
- テキスト: #111827 (Gray 900)
