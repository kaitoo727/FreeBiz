# AI仕訳プロンプトテンプレート

## 経費分類プロンプト

```
以下の経費情報を分析し、適切な会計科目を提案してください。

説明: {description}
金額: {amount}円
{receiptText ? `領収書テキスト: {receiptText}` : ''}

以下の科目から最も適切なものを選択してください：
- 通信費（インターネット、電話代など）
- 仕入れ（商品の仕入れ）
- 研修費（セミナー、書籍など）
- 交通費（電車、タクシーなど）
- 交際費（飲食代など）
- 消耗品費（文房具、備品など）
- 減価償却費（高額な備品など）
- その他

JSON形式で返答してください：
{
  "category": "科目名",
  "description": "整理された説明文"
}
```

## 仕訳生成プロンプト（青色申告）

### 売上の仕訳

```
売上が発生しました。複式簿記の仕訳を作成してください。

金額: {amount}円
日付: {date}
説明: {description}
カテゴリ: {category}

以下の形式で仕訳を作成してください：
- 借方: 現金/預金
- 貸方: 売上
- 金額: {amount}円
```

### 経費の仕訳

```
経費が発生しました。複式簿記の仕訳を作成してください。

金額: {amount}円
日付: {date}
科目: {category}
説明: {description}

以下の形式で仕訳を作成してください：
- 借方: {category}
- 貸方: 現金/預金
- 金額: {amount}円
```

## OCR結果の改善プロンプト

```
以下のOCRで読み取った領収書テキストを整理し、構造化された情報を抽出してください。

OCRテキスト:
{ocrText}

以下の情報を抽出してください：
- 店舗名
- 金額
- 日付
- 商品・サービス名
- その他の重要な情報

JSON形式で返答してください：
{
  "store_name": "店舗名",
  "amount": 金額,
  "date": "日付",
  "items": ["商品1", "商品2"],
  "other_info": "その他の情報"
}
```

## 減価償却判定プロンプト

```
以下の経費情報を分析し、減価償却の対象かどうかを判定してください。

金額: {amount}円
科目: {category}
説明: {description}

減価償却の対象となる条件：
- 10万円以上の備品・工具・器具
- 使用可能期間が1年以上
- 事業用の資産

JSON形式で返答してください：
{
  "is_depreciation": true/false,
  "reason": "判定理由"
}
```

## 副業バレ防止チェックプロンプト

```
以下の情報を基に、副業バレ防止のためのチェックを行ってください。

事業タイプ: {businessType}
給与所得: {salaryIncome}円
事業所得: {businessIncome}円
申告タイプ: {taxFilingType}

以下の観点でチェックしてください：
1. 普通徴収設定が推奨されるか
2. 入金口座が適切か
3. 会社規則に抵触しないか
4. 所得分離が適切か

JSON形式で返答してください：
{
  "warnings": ["警告1", "警告2"],
  "recommendations": ["推奨事項1", "推奨事項2"]
}
```
